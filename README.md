🧮 Python Calculator

📌 Description

This is a simple command-line calculator built using Python. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

🚀 Features

Addition of two numbers
Subtraction of two numbers
Multiplication of two numbers
Division with error handling (prevents division by zero)
User input validation

🛠️ How It Works

The program displays a menu of operations.
The user selects an operation (1–4).
The user enters two numbers.
The program performs the selected calculation and displays the result.

▶️ Example

Calculator
Choose Operation :
1. Add
2. Subtract
3. Multiply
4. Divide

Enter your choice between (1/2/3/4): 1
Enter your first number here: 5
Enter your second number here: 3
Result = 8

⚠️ Error Handling

If the user selects an invalid option, the program shows an error message.
Division by zero is handled using try-except.
